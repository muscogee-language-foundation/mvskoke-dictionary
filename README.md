# mvskoke_dictionary

Mvskoke language dictionary using the [Mother Tongues Dictionaries](https://github.com/MotherTongues/mothertongues) format.  

[Data repository](https://github.com/muscogee-language-foundation/mtd-starter) 

## Installation
- Install [Flutter](https://docs.flutter.dev/install)  
- run the app with `flutter run` 

## To update the data
- Follow instructions in the [dictionary-data](https://github.com/muscogee-language-foundation/dictionary-data) repo to build and export a new `dictionary_data.json` file
- replace the updated file into `assets/`
